\section{实验八：跨音速压气机失速/喘振虚拟实验平台}

\subsection{实验目的}

\begin{enumerate}
    \item 深入了解跨音速压气机失速和喘振的机理。
    \item 掌握跨音速压气机稳态和动态测量方法。
    \item 理解B参数失稳判据及其在失速/喘振判别中的应用。
    \item 观察失速团的周向传播现象和喘振循环的轴向脉动特征。
\end{enumerate}

\subsection{实验平台简介}

本虚拟实验平台为西北工业大学跨音速压气机失速/喘振虚拟实验平台，基于动力能源学院叶轮机械精细化研究团队多年科研积累开发。平台为一套单级跨音速压气机，最高转速 20000 RPM，流量范围 0\textasciitilde24 kg/s，单级压比 1.86。

虚拟实验核心要素包括高保真模型、虚拟增强场景、数据模型驱动和误差配置，提供全流程在线教学体验。访问方式：登录实验空间（国家虚拟仿真实验教学项目共享服务平台）。

平台基于 M-G 模型（Moore-Greitzer 二维压缩系统理论模型），实验流程和操作仿真度达95\%以上。

\subsection{失速/喘振判别理论}

Moore 和 Greitzer 提出的二维压缩系统理论模型定义了无因次参数 $B$：

\begin{equation}
    B = \frac{U}{2a} \sqrt{\frac{V_p}{A_c L_c}}
\end{equation}

式中 $U$ 为转子切线速度，$a$ 为当地音速，$V_p$ 为出口储气室容积，$A_c$ 为压气机进口流通面积，$L_c$ 为压气机进口管道长度。$B$ 参数越大越容易发生喘振，越小越容易出现旋转失速。

\subsection{实验步骤}

\subsubsection{低压比工况（11000 rpm，$B$ 参数$<$临界值）}

\begin{enumerate}
    \item 点击``实验操作''按钮，学习安全须知后进入实验操作系统。
    \item 检查所有连线正常，依次开启工控机、动态信号分析仪、信号转换器。
    \item 打开压气机特性采集软件，点击``零点采集''完成传感器零点电压采集。
    \item 打开节流锥控制程序，点击``复位''确保节流锥回到起始位置。
    \item 在动力柜中打开电机电源（此时变频器为0 Hz）。
    \item 打开齿轮增速箱润滑油供油阀门和油泵启动按钮。
    \item 在系统界面设置转子转速为11000 rpm，观察转速表逐渐升高到设定值。
    \item 设置节流锥步进量，逐步推进节流锥减小流量，每步待信号稳定后采集数据。
    \item 持续减小流量直至发生旋转失速：压气机性能点突降，失速警报灯闪烁，伴随失速噪声。
    \item 打开``局部特写''窗口和``可视化''按钮，观察失速团的周向传播现象。
    \item 后退节流锥增大流量并采集数据，观察恢复稳定工作时的``迟滞''现象。
    \item 完成该转速测量后点击``结束记录''，点击``复位''。
\end{enumerate}

\subsubsection{中等转速工况（15000 rpm，$B$ 参数$<$临界值）}

\begin{enumerate}
    \item 提升压气机转速至15000 rpm。
    \item 重复节流测量过程，观察该转速下的失速及迟滞现象。
    \item 对比11000 rpm工况，分析转速对失速特性的影响。
\end{enumerate}

\subsubsection{高转速工况（17000 rpm，$B$ 参数$>$临界值）}

\begin{enumerate}
    \item 提升压气机转速至17000 rpm。
    \item 缓慢移动节流锥减小流量并采集数据，直至发生喘振。
    \item 喘振发生时：``喘振''警报灯闪烁，伴随低频大幅喘振噪声和实验台振动增大。
    \item 观察喘振状态下流量与特性参数的周期性波动。
    \item 打开``可视化''按钮观察喘振状态气流轴向大幅脉动现象。
    \item 观察完毕后及时点击``退喘''按钮（若超过5秒系统自动退喘放气）。
    \item 逐渐降低电机转速至零，依次关闭各程序。
\end{enumerate}

\subsection{实验现象与分析}

\subsubsection{旋转失速现象（11000 rpm / 15000 rpm）}

在低$B$参数条件下，随着节流锥推进、流量减小，压气机经历稳定工作阶段。当流量减小到失速边界时：压气机总压升突降，工作点跳至特性曲线另一分支；失速警报灯亮起，伴随高频失速噪声；可视化窗口显示失速团沿周向传播，速度约为转子转速的30\%\textasciitilde50\%；退出失速时存在明显的``迟滞''现象，需将节流锥后退到比进入失速时更大的流量才能恢复稳定。

\subsubsection{喘振现象（17000 rpm）}

在高$B$参数条件下，当流量减小到喘振边界时：压气机发生喘振，性能点在压比-流量图上出现大幅周期性波动；喘振警报灯亮起，伴随低频（数Hz）大幅噪声；整个压缩系统出现轴向大幅气流脉动，可能伴随倒流；喘振对发动机具有严重破坏性，需在5秒内退喘。

\subsection{结论}

通过跨音速压气机失速/喘振虚拟实验平台，安全直观地观察了压气机从稳定工作到旋转失速再到喘振的完整过程。实验验证了 M-G 模型 $B$ 参数判据的有效性：低 $B$ 参数时系统发生旋转失速，高 $B$ 参数时发生喘振。观察到的退出失速时的迟滞现象反映了压缩系统的非线性特性，加深了对压气机失稳机理的理解。
