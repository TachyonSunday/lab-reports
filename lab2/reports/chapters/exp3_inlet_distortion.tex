\section{实验三：进气畸变条件压气机性能实验}

\subsection{实验目的}

\begin{enumerate}
    \item 了解进气畸变对压气机性能和稳定性的影响。
    \item 掌握畸变流场的评价指标和测量方法。
    \item 分析不同畸变形式（周向、径向）对压气机特性退化的差异。
\end{enumerate}

\subsection{实验原理}

在理想状态下，航空发动机压气机进口截面流动均匀。但在实际飞行中，当飞机处于大迎角/大侧滑角飞行、导弹发射吞烟、进气道附面层吸入等情况下，压气机进口会产生严重的不均匀流动——进气畸变。进气畸变是诱发压气机内部流动失稳、导致发动机喘振的关键因素之一。

本实验通过安装不同形式的畸变发生器（畸变插板），人为制造进口总压的不均匀分布，测量畸变条件下的压气机特性，与均匀进气特性对比，量化进气畸变对压气机失稳裕度和增压能力的影响。

常用的畸变评价指标包括周向畸变指数和径向畸变指数，分别描述进口压力场沿周向和径向的不均匀程度。

\subsection{实验装置}

\begin{itemize}
    \item 两级低速轴流对转压气机实验台（同实验一）
    \item 畸变发生器（插板）：
    \begin{itemize}
        \item 30\textdegree~圆心角扇形畸变片（周向畸变）
        \item 75\textdegree~月牙形畸变片（同时含周向和径向不均匀）
        \item 90\textdegree~月牙形畸变片（更严重的综合畸变）
    \end{itemize}
    \item 稳态参数测量系统（进口总压、静压、出口总压）
    \item 节流锥控制系统
    \item 大气温度、压力测量仪表
\end{itemize}

\subsection{实验步骤}

\subsubsection{均匀进气特性测量}

同实验一的标准操作流程，在无畸变条件下测量1200 rpm时10个工况点的特性数据（节流锥位置170\textasciitilde440 mm）。

\subsubsection{畸变进气特性测量}

\begin{enumerate}
    \item 停机，更换进气流量管机匣段，安装畸变发生器插板。
    \item 读取并记录大气温度 $T_{\text{atm}}$、大气压力 $P_{\text{atm}}$。
    \item 重新启动压气机，加速到1200 rpm并等待转速稳定。
    \item 状态稳定后测量各参数：流量管静压 $P_0$、进气总压 $P_1^*$、级后总压 $P_2^*$。
    \item 调节节流锥推进，记录至少5个特性点的数据。
    \item 更换不同形式和尺寸的畸变片，重复步骤\textbf{2-5}。
    \item 实验结束：停止变频器，关闭所有程序，拆除畸变插板。
\end{enumerate}

\subsection{数据处理方法}

质量流量和总压升的计算方法同实验一。对于畸变进气条件，进口总压的参考值取周向平均总压。

本实验重点对比均匀进气与畸变进气条件下的总压升-质量流量特性曲线，分析畸变导致的：
\begin{enumerate}
    \item 总压升的绝对下降量
    \item 失稳边界（近失速点）流量的变化
    \item 稳定工作范围的缩减（失稳裕度的退化）
\end{enumerate}

\subsubsection{结果分析}

\begin{enumerate}
    \item \textbf{总压升降低}：进气畸变导致压气机总压升在整个流量范围内降低。畸变越严重（畸变片覆盖的圆心角越大、形式越复杂），性能下降越明显。90\textdegree~月牙形畸变对应的总压升下降幅度最大。
    \item \textbf{失稳边界提前}：畸变条件下的失稳边界（近失速点）向大流量方向移动，即压气机在更大的流量下就进入不稳定工作状态，稳定工作范围缩小。
    \item \textbf{失稳裕度分析}：进气畸变导致压气机失稳裕度降低的主要机理为：周向不均匀进气使部分叶片通道在转子旋转中周期性地经历高负荷状态（处于畸变低压区的叶片通道有效攻角增大），更容易发生附面层分离，引发旋转失速。
    \item \textbf{畸变形式的影响差异}：月牙形畸变（75\textdegree/90\textdegree）同时包含周向和径向的不均匀性，相比纯周向扇形畸变（30\textdegree），对性能的影响更复杂。月牙形畸变不仅造成低压区叶片的高攻角问题，还引起径向压力梯度驱动的二次流。
    \item \textbf{工程意义}：该实验结果为发动机在实际飞行条件下（如大迎角起飞、机动飞行、武器发射吞烟等）的工作特性评估和稳定性控制提供了基础实验依据。
\end{enumerate}

\subsection{结论}

通过对比均匀进气和三种畸变进气条件下的压气机特性，量化了进气畸变对压气机性能的影响。进气畸变导致总压升降低、失稳边界提前、稳定工作范围缩小，且影响程度与畸变类型和严重度正相关。月牙形畸变（同时含周向和径向不均匀）比纯周向扇形畸变对性能的影响更为严重。这些结论为发动机进气畸变容限设计和防喘控制策略提供了实验支持。
