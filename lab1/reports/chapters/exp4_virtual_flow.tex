\section{实验四：发动机虚拟流动实验}

\subsection{实验目的}

掌握气流在发动机内部运动的特点及变化规律。

\subsection{实验装置}

气流经过某发动机内部流动的视频文件，展示气流在发动机各主要部件中的流动状态，包括压气机流动（正常与失速）、燃烧室流动、涡轮流动、排气混合器和加力燃烧室流动。

\subsection{实验步骤及内容}

\begin{enumerate}
    \item 启动流动显示视频文件。
    \item \textbf{压气机流动观察}：观察气流经过压气机时流道和气流状态的变化，尤其注意观察正常流动和失速的差别。正常流动时气流沿叶栅通道有序流动，压力逐级升高；失速时叶片吸力面出现大面积分离，通道内形成失速团。
    \item \textbf{燃烧室流动观察}：观察气流经过燃烧室时的流动情况，尤其注意内部的涡系结构和温度分布。重点关注头部回流区的形成、旋流器的旋流作用、掺混孔的射流穿透和出口温度分布。
    \item \textbf{涡轮流动观察}：观察气流经过涡轮时的流动情况。气流在导向器中加速膨胀，在转子中将热能转化为机械功。
    \item \textbf{排气混合器与加力燃烧室流动观察}：观察内外涵气流在排气混合器中的掺混过程，以及加力燃烧室中火焰稳定和燃烧组织方式。
\end{enumerate}

\subsection{实验分析与讨论}

\begin{enumerate}
    \item \textbf{压气机正常流动与失速流动的不同}

    正常流动特征：
    \begin{itemize}
        \item 气流沿叶栅通道平稳流动，叶片吸力面附着良好。
        \item 各叶片通道内流场均匀一致，出口压力和速度分布均匀。
        \item 压力沿流向逐级升高，速度分布符合设计意图。
    \end{itemize}

    失速流动特征：
    \begin{itemize}
        \item 叶片吸力面出现大面积分离区，形成低能流体聚集。
        \item 部分叶片通道被堵塞，迫使气流偏转到相邻通道。
        \item 失速团以转子转速的30\%\textasciitilde50\%的速度沿周向传播。
        \item 压气机总压升急剧下降，出口气流出现大尺度脉动。
        \item 严重时发展为喘振，整个压气机系统出现轴向大幅振荡。
    \end{itemize}

    \item \textbf{火焰筒内气流的流动特点以及具有该特点的原因}

    火焰筒内气流具有以下流动特点：

    \begin{enumerate}
        \item \textbf{头部回流区}：经旋流器进入的旋转气流在扩张段产生逆压梯度，形成中心回流区。回流区将高温已燃气体带回火焰筒头部，为新鲜油气混合物提供连续点火源，实现火焰稳定。这是航空发动机燃烧室最核心的稳焰机制。

        \item \textbf{分层进气}：进入火焰筒的空气分为三股——头部进气（经旋流器，约15\%\textasciitilde30\%）、主燃区掺混进气（经主燃孔，约30\%）和掺混/冷却进气（经掺混孔和气膜孔，约40\%\textasciitilde50\%）。这种分层进气设计使得主燃区维持接近化学恰当比的高温燃烧，而掺混区迅速稀释降温以获得要求的涡轮进口温度分布。

        \item \textbf{壁面冷却气膜}：火焰筒壁面布置大量冷却小孔或冷却环带，冷却空气沿壁面形成保护气膜，将壁面与高温燃气（可达2000\textasciitilde2400 K）隔离，保证火焰筒结构完整性。
    \end{enumerate}
\end{enumerate}

\subsection{结论}

通过本次虚拟流动实验，直观地观察了气流在航空发动机各主要部件中的流动状态。掌握了压气机正常流动与失速流动的判别特征、燃烧室火焰筒内回流区稳焰机理和分层进气原理，加深了对《航空发动机原理》和《燃烧学》课程中相关内容的理解。
